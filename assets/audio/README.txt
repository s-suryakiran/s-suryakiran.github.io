Mars easter egg audio
=====================

The portfolio looks for an audio file at:

  assets/audio/mars-botanist.mp3

When the rocket lands on Mars, it tries to play this clip at low volume.
If the file is missing, nothing happens — the site just silently continues.

Drop any clip you have the rights to use here, named exactly
`mars-botanist.mp3`. A short clip (a few seconds) works best.

Supported formats: any format the browser understands. If you want to
use a different filename or format, edit the `MARS_AUDIO_SRC` constant
near the top of `assets/js/space.js`.
